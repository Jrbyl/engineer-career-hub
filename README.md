# Engineer Career Hub

Engineer Career Hub is mission control for the internship search. It connects application tracking to interview readiness with a role-centric pipeline, a structured STAR story bank, resume notes for each role, and an interview pack view.

## Run Locally

```bash
npm start
```

The server starts at `http://localhost:3000`, or the next available port if `3000` is already in use.

## Check Syntax

```bash
npm run check
```

## MVP Scope

- Create, edit, delete, search, and stage applications.
- Save job posting URLs, deadlines, location, notes, and role-specific resume notes.
- Create, edit, delete, search, and tag STAR stories.
- Link stories to an application.
- View and copy a read-only interview pack for a selected role.
- Persist all data in browser `localStorage`.
